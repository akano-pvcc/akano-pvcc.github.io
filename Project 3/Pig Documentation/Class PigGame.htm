Class PigGame

public class PigGame

PigGame - play a game of Pig with the given players

Output format:

  • Before each turn, prints a line with "Player 1 score: " and player 1's score. Prints another line with "Player 2 score: " and player 2's score. Finally, prints a line with "It is player #'s turn.", where "#" is replaced by the current player number. Play starts with player 1 and then alternates.
  • For each roll, print a line with "Roll: " and the random die roll value (1-6).
  • After each non-"pig" roll, test whether the given player wishes to roll/hold, and continue/end the turn accordingly.
  • After a "pig" roll of 1, or a "hold", print a line with "Turn total: " followed by the turn total. In the case of a "pig", this turn total is 0. Then, print a line with "New score: " followed by the new score for the current player.
